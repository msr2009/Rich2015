:py:mod:`~seqlib.basic` --- Single-read sequencing library
==========================================================

.. py:module:: seqlib.basic
	:synopsis: Single-read sequencing library.

The :py:mod:`~seqlib.basic` module contains the class definition for the :py:class:`~seqlib.basic.BasicSeqLib`, which stores data from a sequencing library constructed to have a single read (forward or reverse) for each variant sequence.
